## About the project:

### Topic
	Website Contability-learning platform.
### Database entities

	- User:
	
		- Anonymous.
		- Student.
		- Teacher (administrator).
		
	- Unit.
	- Lesson.
	- Exercise.
	- Answer.
	
### Advanced functionality

	- Progress Statistics.
	- Basic User Information.
	
### Graphic information
	We are going to use line graphics showing students information (days/hours).	
### Other functionality

	- Logging alternatives.
	- Export to excel.
	- Send e-mails.
